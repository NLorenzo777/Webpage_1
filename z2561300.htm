AP30 Parallel Interface Board (PIB) Firmware Upgrade

Note: Firmware upgrades to V1.22 and V1.30 requires PIB V1_01 firmware. Firmware upgrades to DSP V1.32 and higher require PIB V1_02 firmware.
The GlobalSerialFlash is used to upgrade the PIB. The update itself is quick and easy to accomplish.
Tools:
  • Service computer with appropriate Global Serial Flash installed.
  • Communications cable - RS232 straight-through serial cable.
A new version of the Firmware Update Utility was released in 2017 in support of the EXL S1.

Flash Update Procedure

Note: If maintenance bypass is not available, this procedure will require the customer load to be shut down.
How to Use the Checkboxes: The checkboxes in the following procedure are meant to help track progress. After completing a step, check tbe procedure off by ticking the corresponding checkbox. The check mark(s) will be saved, so that leaving the page is possible and the user can continue with the procedure. To clear all check marks and remove saved progress on this page, click here.
  1. Continue with a serial cable (same cable and port used in DSP Firmware Update) connected from the service laptop to the female DB9 connector XP6-AP30-XP3 located on top of the unit.
  2. Place the XP30-XP99 SWs (1 and 2) to the ON position.
  3. Launch the Flash Update (FU) Utility application (refer to Global Serial Flash).
  4. Select the following GlobalSerialFlash menu settings:
    • Product: EXL S1
    • Device: Parallel Interface Board Tiva M4 MCU
    • COM Port: PC COM Port
    • Application File: Vx_xx_PIB_S1_Tiva.bin
    • Reference Figure 1.0
    Note: The programming instructions are on the right side of the application window.
    Note: The image shown is for reference use only.
  5. Click Start to begin programming.
  6. Close the external input circuit breaker or external rectifier breaker if the unit has a dual-fed input.
  7. When completed, exit the Flash Update (FU) Utility (GlobalSerialFlash.exe) application.
  8. Open the external input circuit breaker or external rectifier breaker if the unit has a dual-fed input.
  9. Return the PIB flash SWs XP30-XP99 (1 and 2) to their original positions.
  10. Remove the serial cable from the laptop to DB9 connector XP6-AP30-XP3.
